# 英语单词

> 单词：前缀 - 词根 - 后缀

- 前缀：对词根意思加以改变 [Un - 在形容词前表示否定 ‘不，无’]
- 词根：构成单词的基本意思 [hap - 机会，运气]
- 后缀：不变词义只变词性 [ness - 一般构成抽象名称，愿意不变]
